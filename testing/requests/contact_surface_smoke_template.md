# Contact Surface Smoke Request Template

## Metadata

- request_id: `contact-surface-smoke-YYYY-MM-DD`
- target_env: `staging-or-production`
- module: `homepage`
- browser_execution: `agent-browser`
- model: `zhipuai-coding-plan/glm-5`

## Goal

验证首页 Contact 区块的可见性、邮箱入口与对外联系表面是否清晰可用。

## Checks

1. Contact 区块是否可见
2. 联系邮箱是否显示正确
3. 至少 1 个联系入口是否可点击
4. 文案是否仍保持当前品牌口径

## Evidence Requirements

1. Contact 区块截图
2. 邮箱文本证据
3. 点击结果或链接目标证据
4. 如失败，写明是文案问题、链接问题还是布局问题
