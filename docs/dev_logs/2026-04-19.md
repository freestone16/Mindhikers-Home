## 本轮新增治理结论

已写入 `docs/rules.md`：

1. WP 与 Next.js 的 revalidate tag 必须逐字一致，且集中维护
2. `/api/revalidate` 必须 POST-only
3. tag 必须走 allowlist / 明确前缀校验
4. non-blocking webhook 也必须留下可检索日志
5. 前台接 WordPress 图片时，必须同步核对 `remotePatterns` 与 CSP `img-src`
